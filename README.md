# Dotfiles

Central repository for macOS configuration files.

## Structure

- `karabiner/` – keyboard logic
- `zsh/` – shell config
- `vscode/` – editor settings
- `iterm2/` – terminal profiles
- `brew/` – installed packages snapshot

Each subfolder contains its own README with setup instructions.

## Installation on new Mac

Clone repository:

```bash
git clone git@github.com:YOUR_USERNAME/dotfiles.git ~/git/dotfiles
